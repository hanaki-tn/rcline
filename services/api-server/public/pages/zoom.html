参加手順

  1. 上の「Zoomアプリで参加」または「ブラウザで参加」をタップします。
  2. アプリが起動しない場合は、下のミーティングIDとパスコードを手動で入力してください。
  3. 初回は音声やカメラのアクセス許可を求められる場合があります。

トラブル時の確認ポイント

  • マイクやカメラが正しく選択されているか
  • 音量がミュートになっていないか
  • インターネット接続が安定しているか
  • 音声や映像がうまく接続できない場合は、下記「アクセス方法（公式ガイド）」を参照してください。
